# 专题目录命名

专题目录就是重构包文件夹名称，使用 a+日期+专题英文或拼音+版本 统一使用小写字母，例如：a20090817avatar12，文件名禁止以数字开头。 重构包命名优先使用有意义的英文名，如使用拼音，拼音字符超过10个，请使用首个拼音的全称+后面拼音的简写
